# Atlas - 3D Open-World Procedural Game

A procedurally generated open-world 3D game built with Unity and C++.

## Features
- Procedural world generation using C++
- Open-world exploration
- Dynamic terrain generation
- Real-time rendering with Unity

## Project Structure
```
atlas.game/
├── Assets/           # Unity assets (models, textures, sounds)
├── Source/          # C++ source code for procedural generation
├── Scripts/         # C# Unity scripts
├── Docs/            # Documentation
├── Build/           # Build output
└── CMakeLists.txt   # CMake configuration for C++
```

## Requirements
- Unity 2022 LTS or higher
- C++ 17 or higher
- CMake 3.20+

## Getting Started
1. Clone the repository
2. Open with Unity
3. Build C++ modules with CMake
